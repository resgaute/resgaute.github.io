Dúvidas

Perguntas frequentes

Respostas diretas para você doar com segurança.

Quando faço um PIX, para onde vai o dinheiro?
Vai direto para a conta oficial e é usado no essencial (ração, remédios, cuidados e manutenção).
Posso acompanhar prestação de contas?
Sim. Você pode acompanhar atualizações e, quando possível, comprovantes/relatórios.
Minha doação é segura?
Sim. O PIX é feito no seu banco. Este site apenas facilita a cópia do código e o QR Code.
Se eu não puder doar, como ajudo?
Compartilhar já ajuda muito. Envie a mensagem pronta para 3 pessoas.
